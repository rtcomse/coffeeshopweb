About

This is a coffee shop website.

Users can place an order for a cup of coffee.

Users can customize their coffee order by selecting from a variety of ingredients.

Users can view the history of their coffee orders.


The navigation bar at the top can be used to visit the Order Page by clicking on "Order".


The navigation bar at the top can be used to visit the History Page by clicking on "History".

Order Page

The Order Page lists all the ingredients that the user can add to their coffee.

The user is able to select all the ingredients for their coffee order and submit the order.

History Page

The History Page lists the past coffee orders made by the user.

The user can select one of the coffee orders to view the list of ingredients that were used.
